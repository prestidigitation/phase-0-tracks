  • Coati-Mundi/coatimundi
  • hog-nosed coon
  • tejón

  • moncún

Coatis are being over-hunted and their environments are being destroyed in Central and South America. There is currently a lack of data on coati population numbers so the impact of deforestation on coatis might be greater than is commonly thought.

What is the purpose of HTML5 semantic tags? Which ones do you see as most useful?

The purpose of HTML5 semantic tags is to provide additional meaning to the information on websites, taking it a step further than only defining the presentation of the info. For example, using the tag <i> has been somewhat deprecated as a way of italicizing text, and the use of <em> for emphasis has taken its place. In this way, the CSS file can define exactly how text should be emphasized on particular pages, so the focus is put on the semantic meaning that certain info is important rather than mere styling. This semantic meaning provides additional avenues for web crawlers to make sense of web sites. I see the <header>, <nav>, and <footer> tags as being particularly useful since most websites with multiple pages can be improved by using uniform navigational features between pages.

What is the order the browser uses to display elements on a page?

The browser parses HTML from top to bottom, left to right, and thus the default order is displaying elements in the order they are put in HTML. For elements in the same level of a containing block, block elements are stacked vertically on top of one another and inline elements are arranged left to right.
